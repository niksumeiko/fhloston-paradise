# Fhloston Paradise
A programming kata in a form of a simple React SPA with a login flow via JWT.

Used for training purpose within [Better Engineering Community](https://www.withnik.com/community) founded by Nik Sumeiko.

## Prerequisites
- Node.js 18+
- npm

## Getting started
```bash
npm install
npm start
```

This starts both the API server (http://localhost:3001) and the React app (http://localhost:5173) in parallel.
